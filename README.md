# Bellabeat Case Study

## Overview

This project is a case study conducted as part of the Google Data Analytics Professional Certificate on Coursera.
The analysis uses publicly available Fitbit fitness tracker data from Kaggle to explore user behavior patterns and provide data-driven recommendations for improving user engagement and long-term product usage.

## Business Task

The objective of this analysis is to identify behavioral patterns among smart device users and provide actionable recommendations to support Bellabeat's marketing and product development strategies.

## Data Source

The dataset used in this analysis is the Fitbit Fitness Tracker Data, which is publicly available on Kaggle.

- Source: Kaggle
- Dataset: [FitBit Fitness Tracker Data](https://www.kaggle.com/datasets/arashnic/fitbit)
- License: Public Domain

This dataset includes daily activity, sleep, and device usage records collected from Fitbit users.

## Key Findings

- Consistent Active Users demonstrated the highest physical activity levels, exceeding 9,500 daily steps with the lowest sedentary time.
- Irregular Users showed moderate activity levels but the highest variability in daily behavior.
- Low Engagement Users recorded the lowest activity levels and the highest sedentary time, exceeding 1,200 minutes.
- Moderate Users maintained consistent device usage and stable daily behavior patterns.

## Business Recommendations

- Implement a gamification system to encourage users to reduce sedentary time.
- Provide educational content to promote sustainable habit formation.
Focus on long-term engagement strategies rather than short-term promotions.
